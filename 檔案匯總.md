## 各檔案摘要

### 期中.md — SHOPLINE 一頁式購物網站
- **技術架構**：前端 HTML+CSS+JS（無框架），後端 Node.js+Express 5，資料庫 SQLite
- **資料庫**：3 張表（`products`、`orders`、`order_items`）
- **API**：`GET /api/products`（商品列表，支援分類篩選）、`POST /api/orders`（建立訂單）
- **功能**：商品分類篩選、購物車抽屜、結帳彈窗、Toast 提示、RWD

### 練習4.md — JavaScript 基礎練習（10 題）
- 主題：if 判斷、for/while 迴圈、function、Array、Object、JSON
- 綜合挑戰：學生成績管理系統（Array + Object + Function + JSON）

### 練習5.md — 簡易網誌系統開發紀錄
- **技術**：Next.js 16 + TypeScript + Tailwind CSS + SQLite（better-sqlite3）
- **三階段開發**：
  1. 專案初始化與文章 CRUD
  2. 登入系統（bcryptjs + jose JWT）
  3. 個人/公開貼文區
- **路由**：`/`（公開時間軸）、`/login`、`/register`、`/posts/new`、`/posts/[id]/edit`、`/my-posts`、`/users/[id]`
